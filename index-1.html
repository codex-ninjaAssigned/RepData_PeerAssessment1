6. Code to describe and show a strategy for imputing missing data

The missing values for a particular interval is replaced by the mean of that interval.

r missing_activity<-data[is.na(data$steps),] nonMissing_activity<-data[!is.na(data$steps),] impu_activity<-left_join(missing_activity,tf,by="interval")%>%select(-steps.x)%>%rename(steps=steps.y) cor_data<-rbind(nonMissing_activity,impu_activity)
